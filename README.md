# Expense Tracker App
A simple expense tracker to help you keep track of your daily expenses.

![alt text](https://i.imgur.com/LRAZOW9.gif)

## Installation and Usage
```
yarn install 
```
Run with the following command:
```
yarn run dev-server
```

## Built With
- JavaScript
- React and Redux
- Firebase
- SAAS
- Babel and Webpack
